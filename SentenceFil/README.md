# SentenceFil

## Back End

### Install

```
pip install fastapi
pip install mariadb
pip install uvicorn
```

### Run

#### Create

```
python main.py create
```

#### Run

```
python main.py start
```

### References

https://www.mdbg.net/chinese/dictionary?page=cc-cedict

## Front End (Web)

```
mv -f front front.0
npx create-react-app front
mv -f front.0/src/* front/src/

cd front
npm start
```
